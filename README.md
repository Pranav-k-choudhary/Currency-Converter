# Currency Converter
A simple and responsive Currency Converter Web Application that provides real-time exchange rates using an API.

# Features
- Convert currencies in real-time
- Supports multiple international currencies
- Dynamic country flags update
- Fast and responsive UI
- Mobile-friendly design

# Tech Stack
- HTML – Structure
- CSS – Styling & Responsive Design
- JavaScript – Logic & API Handling

# API Used
- Exchange Rate API:
  https://latest.currency-api.pages.dev/v1/currencies
  
- Flags API:
  https://flagsapi.com

# How It Works
- User enters amount
- Selects "From" and "To" currency
- App fetches real-time exchange rate from API
- Displays converted result instantly

# Concepts Used
- DOM Manipulation
- Async/Await
- Fetch API
- Event Handling
- Dynamic UI Update
